## The Walk Through

1. First download the zip file provided in the in the story page, then extract it somewhere of your choosing.
    
2. The 2 included files mentioned above will be found in the folder named:
    
    > CTF001\_Neurocracked
    
3. Open a terminal and navigate to the folder in step 2.
    
4. We were given clues in the tips, so lets run this command in the terminal:
    
    ```bash
    strings brainos_v14.2_patch.img > brainos_strings.txt
    ```
    
    This is our string based approach analyzing the *brainos\_v14.2\_patch.img* file. This command produces a text file named *brainos\_strings.txt that contains all the printable characters found in the .img file.*
    
5. Next will use grep to search for *‚ÄúECHO\_WORDS"*, from the instructions. Run this command in the terminal:
    
    ```bash
    grep "ECHO_WORDS" brainos_strings.txt
    ```
    
    This will give us the full payload string needed to continue.
    
6. The instructions state we need to submit a SHA-256 hash, so this is the next command to run:
    
    ```bash
    echo -n "ENTER FULL PAYLOAD STRING HERE" | sha256sum
    ```
    
    You will need to replace ‚Äú*ENTER FULL PAYLOAD STRING HERE*‚Äù with the actual payload string found in step 5.
    
7. Next head over to the Neurocracked CTF portal, and submit the hash produced in step 6:
